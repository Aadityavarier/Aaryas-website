## 🎐 About the Project
Welcome to a slice of the internet designed with a bespoke **Sakura × Samurai** aesthetic. This portfolio highlights Aarya's personality, interests, and achievements through a beautiful, modern interface.

### ✨ Highlights
- ⚡ **Modern & Fast**: Scaffolded with **React** and **Vite** for blazing-fast load times.
- 🎨 **Deep Customization**: A rich aesthetic employing custom color palettes (warm sakura pinks, deep void blacks, and subtle gold accents).
- 🪄 **Fluid Interactions**: Immersive animations, 3D tilt-responsive cards, magnetic cursor buttons, and gently falling Sakura petals.
- 🧩 **Dynamic Displays**: Dedicated sections for About, Skills, Hobbies, Favorites, and Education.

## 🚀 Getting Started

Want to spin this up locally and experiment? Just follow these steps!

1. **Clone the repository:**
   ```bash
   git clone https://github.com/Aadityavarier/Aaryas-website.git
   ```
2. **Navigate into the directory:**
   ```bash
   cd Aaryas-website
   ```
3. **Install dependencies:**
   ```bash
   npm install
   ```
4. **Start the development server:**
   ```bash
   npm run dev
   ```
